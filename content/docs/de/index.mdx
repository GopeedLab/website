Gopeed (vollständiger Name Go Speed) ist ein Hochgeschwindigkeits-Downloader, der mit `Golang` + `Flutter` entwickelt wurde. Er unterstützt `HTTP`, `BitTorrent`, `Magnet` und `ed2k` Downloads und ist auf allen Plattformen verfügbar, darunter:

- [x] **Windows**
- [x] **Macos**
- [x] **Linux**
- [x] **Android**
- [x] **iOS**
- [x] **Web**
- [x] **Docker**
- [x] **Kommandozeile**

## Funktionen

Als moderner Downloader bietet Gopeed viele Funktionen. Hier ist eine kurze Übersicht.

### Grundlegend

- Unterstützung für HTTP & HTTPS
- Unterstützung für BitTorrent, Magnet & ed2k
- Seed, DHT, PEX, uTP, Webtorrent, Upnp
- Automatische tägliche Aktualisierung der Tracker-Liste
- Internationalisierung
- Dunkelmodus

### Erweitert

Zusätzlich zu den grundlegenden Download-Funktionen gibt es viele erweiterte Funktionen, die Gopeed noch vielseitiger machen.

- **Offene HTTP-API**

  Über die RESTful-API können Sie problemlos mit Gopeed interagieren. Beispielsweise können Sie Downloads starten, pausieren, löschen und weitere Aktionen über die API steuern.

- **Dezentrale Erweiterungen**

  Sie können Erweiterungen in `JavaScript` schreiben, um die Download-Funktionen von Gopeed zu erweitern, z. B. das Herunterladen von Videos oder Musik von einer Website.
