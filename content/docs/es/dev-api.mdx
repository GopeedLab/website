Gopeed proporciona una interfaz API HTTP para uso externo, lo que permite la gestión de descargas a través de la API.

## Habilitar la API

Primero, necesitas configurar el protocolo de comunicación a TCP. Ve a **Configuración → Avanzado → Protocolo de comunicación** y establece el protocolo de comunicación a TCP. Luego, configura la IP y el puerto como se muestra en la siguiente imagen:

![](/images/dev/set-port.png)

<Callout type="info">
  Puedes configurar cualquier `port`, pero ten cuidado de no usar puertos que ya
  estén en uso o reservados por el sistema, ya que esto puede causar fallos de acceso.
</Callout>

Después de eso, puedes acceder a la API a través de `http://127.0.0.1:9999`. Por razones de seguridad, se recomienda establecer un token. Ve a **Configuración → Avanzado → API Token** y configura un token como se muestra en la siguiente imagen:

![](/images/dev/set-token.png)

> Nota: Los ajustes anteriores surten efecto después de reiniciar.

## Usar el SDK JS

Gopeed proporciona una biblioteca JS oficial que debe instalarse primero ejecutando `npm install @gopeed/rest`:

```
npm install @gopeed/rest
```

Luego puedes usarla fácilmente, por ejemplo:

```js
import { Client } from "@gopeed/rest";

(async function () {
  // Crear un cliente
  const client = new Client();
  // Crear una tarea
  const res = await client.createTask({
    req: {
      url: "https://example.com/file.zip",
    },
  });
})();
```

Más usos, código fuente y actualizaciones del paquete están disponibles en el [repositorio oficial de Gopeed JS](https://github.com/GopeedLab/gopeed-js).
