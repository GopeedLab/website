Gopeed (nombre completo Go Speed) es un gestor de descargas de alta velocidad desarrollado con `Golang` + `Flutter`, que soporta descargas `HTTP`, `BitTorrent`, `Magnet` y `ed2k`, y es compatible con todas las plataformas, incluyendo:

- [x] **Windows**
- [x] **Macos**
- [x] **Linux**
- [x] **Android**
- [x] **iOS**
- [x] **Web**
- [x] **Docker**
- [x] **Herramienta de línea de comandos**

## Características

Como gestor de descargas moderno, Gopeed tiene muchas funcionalidades. A continuación, una breve introducción.

### Básicas

- Soporte para HTTP y HTTPS
- Soporte para BitTorrent, Magnet y ed2k
- Seed, DHT, PEX, uTP, Webtorrent, UPnP
- Actualización automática de la lista de trackers todos los días
- Internacionalización
- Modo oscuro

### Avanzadas

Además de las funciones básicas de descarga, hay muchas funciones avanzadas que hacen que Gopeed sea más versátil.

- **API HTTP abierta**

  A través de la API RESTful, puedes interactuar fácilmente con Gopeed. Por ejemplo, puedes controlar las descargas, pausas y eliminaciones de Gopeed a través de la API.

- **Extensión descentralizada**

  Puedes escribir extensiones usando `JavaScript` para mejorar las capacidades de descarga de Gopeed, como descargar vídeos de un sitio web o descargar música, etc.
