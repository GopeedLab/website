Los siguientes métodos funcionan en todas las plataformas para descargar Gopeed rápidamente:

1. Ve al [sitio web oficial](https://gopeed.com) para descargarlo. El sitio seleccionará automáticamente la versión correspondiente según tu sistema operativo.
2. Ve a [GitHub](https://github.com/GopeedLab/gopeed/releases) para descargarlo. Puedes elegir la versión que desees descargar manualmente.

## Guía de instalación por plataforma

### Windows

Windows tiene dos distribuciones: una versión portable y una versión de instalación. La versión portable no necesita instalación, simplemente descomprímela y ejecútala. La versión de instalación necesita ser instalada, y el proceso de instalación es el mismo que el de otros programas. Puedes elegir según tus preferencias.

> El nombre del archivo de la versión portable es: Gopeed-v1.x.x-windows-amd64-portable.zip

Además de la descarga directa, también puedes instalar o actualizar Gopeed a través de gestores de paquetes:

- [Scoop](https://github.com/ScoopInstaller/Scoop): Un gestor de paquetes de línea de comandos mantenido por la comunidad para Windows. Es ligero y flexible, no requiere privilegios de administrador para instalarse, y facilita la gestión y actualización de software.

  ```powershell
  scoop bucket add extras
  scoop install extras/gopeed
  ```

- [Winget](https://learn.microsoft.com/en-us/windows/package-manager/winget): El gestor de paquetes oficial de Microsoft, profundamente integrado con el sistema operativo Windows.

  ```powershell
  winget install -e --id monkeyWie.Gopeed
  ```

#### Falsos positivos del cortafuegos

En el sistema Windows, algunos programas antivirus pueden detectar falsamente dos componentes principales de Gopeed: `host.exe` para la comunicación con la extensión del navegador y `updater.exe` para las actualizaciones de la aplicación. Para garantizar que la extensión del navegador pueda interceptar correctamente las tareas de descarga y que la aplicación pueda actualizarse automáticamente, añade el `directorio de instalación de Gopeed` a la lista blanca de tu cortafuegos o programa antivirus.

<Callout type="info">
**Descarga segura**: Asegúrate de descargar Gopeed utilizando los métodos descritos en este documento, ya sea desde el [sitio web oficial](https://gopeed.com), el [repositorio oficial de GitHub](https://github.com/GopeedLab/gopeed/releases), o a través de Scoop o Winget. Evita usar fuentes de terceros para garantizar la seguridad del software.

**Sobre los falsos positivos**: Esto puede estar relacionado con las características de las aplicaciones compiladas con el lenguaje Go, y actualmente no hay una buena solución. En el futuro, consideraremos migrar los componentes a desarrollo con Rust para mejorar este problema.

</Callout>

### Linux

En el sistema Linux, proporcionamos paquetes `.deb` y `.AppImage` para la instalación, para que puedas elegir el que mejor se adapte a tus necesidades. Además, también puedes instalar rápidamente usando los siguientes comandos:

- Flatpak

  ```sh
  flatpak install flathub com.gopeed.Gopeed
  ```

- Snap

  ```sh
  sudo snap install gopeed
  ```

### macOS

La versión de macOS proporciona un archivo `.dmg` que se puede instalar haciendo doble clic. El paquete de instalación soporta dos arquitecturas: `intel` y `apple silicon`.

> Consejo: Si la apertura de macOS falla, ejecuta el comando `xattr -d com.apple.quarantine /Applications/Gopeed.app` en la terminal.

### Android

La versión de Android proporciona un archivo `.apk` que se puede descargar e instalar directamente, y soporta todas las arquitecturas de CPU.

### iOS

Actualmente, solo se proporciona el archivo `.ipa` para la plataforma iOS, que necesita ser firmado e instalado por tu cuenta. Se recomienda usar [TrollStore](https://github.com/opa334/TrollStore) para la instalación.

> ¿Por qué no está en la App Store?
>
> Debido al mecanismo de revisión de Apple, las aplicaciones relacionadas con el protocolo BitTorrent no están permitidas.
>
> ¿Por qué no está en TestFlight?
>
> ¡Por falta de presupuesto! El proyecto se mantiene con pura pasión y esfuerzo voluntario. Si se obtiene suficiente patrocinio, se publicará en TestFlight inmediatamente. Así que por favor, apóyanos más. El código abierto no es fácil. ¡Gracias!

### Web

Si necesitas un servicio de descarga remota, puedes considerar usar la versión Web. Gopeed proporciona versiones Web para varias plataformas. Puedes descargarlas según tu sistema y arquitectura de CPU.

![web](/images/guide/install/web.png)

Aquí tienes un ejemplo de cómo usar la versión Web en la plataforma Windows. El uso en otras plataformas es similar.

1. Descarga la versión Web y, después de descomprimir, obtendrás una carpeta. Colócala donde desees.
2. Abre la terminal en el directorio raíz de la carpeta y ejecuta `./gopeed.exe`. Si se ejecuta correctamente, verás la siguiente salida:

   ```bash
      _______   ______   .______    _______  _______  _______
    /  _____| /  __  \  |   _  \  |   ____||   ____||       \
    |  |  __  |  |  |  | |  |_)  | |  |__   |  |__   |  .--.  |
    |  | |_ | |  |  |  | |   ___/  |   __|  |   __|  |  |  |  |
    |  |__| | |  `--'  | |  |      |  |____ |  |____ |  '--'  |
    \______|  \______/  | _|      |_______||_______||_______/

    Server start success on http://[::]:9999
   ```

3. Abre el navegador y visita `http://localhost:9999`.

#### Configuración Web

La versión Web soporta configuración a través de parámetros de línea de comandos o archivos de configuración. Los parámetros de línea de comandos se pueden consultar con `./gopeed.exe -h`:

```sh
$ ./gopeed.exe -h
Usage of C:\Users\levi\Downloads\gopeed-web-v1.3.13-windows-amd64\gopeed.exe:
  -A string
        Bind Address (default "0.0.0.0")
  -P int
        Bind Port (default 9999)
  -T string
        API token, that can only be used when basic authentication is enabled.
  -c string
        Config file path (default "./config.json")
  -d string
        Storage directory
  -p string
        HTTP Basic Auth Password
  -u string
        HTTP Basic Auth Username (default "gopeed")
```

También soporta configuración a través de archivos de configuración. Crea un archivo `config.json` en el directorio raíz con el siguiente contenido:

```json
{
  "address": "",
  "port": 0,
  "username": "",
  "password": "",
  "apiToken": "",
  "storageDir": "",
  "whiteDownloadDirs": ["/root/downloads", "/root/dir/*", "/root/dir?abc"]
}
```

También soporta configuración a través de variables de entorno, con la regla `GOPEED_ConfigKey`, por ejemplo:

```sh
export GOPEED_ADDRESS="0.0.0.0"
export GOPEED_PORT="9999"
export GOPEED_USERNAME="gopeed"
export GOPEED_PASSWORD="xxx"
export GOPEED_APITOKEN=""
export GOPEED_STORAGEDIR=""
export GOPEED_WHITEDOWNLOADDIRS="/root/downloads,/root/dir/*,/root/dir?abc"
```

<Callout type="warn">
  Si estás desplegando en una IP pública, asegúrate de habilitar la autenticación
  de identidad, de lo contrario habrá riesgos de seguridad.
</Callout>

#### Configuración de descarga predeterminada

Si necesitas establecer la configuración de descarga predeterminada cuando el servidor se inicia por primera vez, puedes añadir el campo `downloadConfig` en `config.json`. Para más detalles sobre la configuración, consulta [DownloaderStoreConfig](https://pkg.go.dev/github.com/GopeedLab/gopeed/pkg/base#DownloaderStoreConfig). Aquí tienes un ejemplo:

```json
{
  "address": "127.0.0.1",
  "port": 9999,
  "downloadConfig": {
    "downloadDir": "d:/test"
  }
}
```

### Docker

Un solo comando:

```sh
docker run --name gopeed -d -p 9999:9999 liwei2633/gopeed
```

Montar el directorio de descargas

```sh
docker run --name gopeed -d -p 9999:9999 \
    -v /path/to/download:/app/Downloads \
    liwei2633/gopeed
```

Montar el directorio de datos

```sh
docker run --name gopeed -d -p 9999:9999 \
    -v /path/to/download:/app/Downloads \
    -v /path/to/storage:/app/storage liwei2633/gopeed
```

Especificar el ID de grupo y usuario del contenedor

```sh
docker run --name gopeed -d -p 9999:9999 \
    -e PGID=100 \
    -e PUID=1000 \
    liwei2633/gopeed
```

Si necesitas habilitar la autenticación de identidad, puedes pasar parámetros de variables de entorno:

```sh
docker run --name gopeed -d -p 9999:9999 \
    -e GOPEED_USERNAME="admin" \
    -e GOPEED_PASSWORD="123" \
    -v /path/to/download:/app/Downloads \
    -v /path/to/storage:/app/storage \
    liwei2633/gopeed
```

### Línea de comandos

La versión de línea de comandos requiere un entorno `Golang`. Si no tienes un entorno `Golang` instalado, puedes consultar [aquí](https://golang.org/doc/install) para instalarlo.

Comando de instalación:

```sh
go install github.com/GopeedLab/gopeed/cmd/gopeed@latest
```

Después de la instalación, puedes usar el comando `gopeed` para descargar. Puedes ver el uso específico con `gopeed -h`:

```sh
$ gopeed -h
Usage of gopeed:
  -C int
        Concurrent connections. (default 16)
  -D string
        Store directory. (default "C:\\Users\\levi")
```

#### Ejemplos de uso de la línea de comandos

Descargar un recurso HTTP:

```sh
gopeed https://example.com/file.zip
```

Descargar un archivo torrent:

```sh
gopeed D:/Downloads/file.torrent
```

Descargar un enlace magnet:

```sh
gopeed magnet:?xt=urn:btih:xxxx
```

Descargar un enlace ed2k:

```sh
gopeed 'ed2k://|file|example.iso|123456789|0123456789ABCDEF0123456789ABCDEF|/'
```
