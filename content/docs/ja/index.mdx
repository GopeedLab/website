Gopeed（正式名称 Go Speed）は、`Golang` + `Flutter` で開発された高速ダウンローダーです。`HTTP`、`BitTorrent`、`Magnet`、`ed2k` ダウンロードに対応し、以下のすべてのプラットフォームをサポートしています：

- [x] **Windows**
- [x] **Macos**
- [x] **Linux**
- [x] **Android**
- [x] **iOS**
- [x] **Web**
- [x] **Docker**
- [x] **Command Tool**

## 特徴

モダンなダウンローダーとして、Gopeedには多くの機能があります。ここで簡単にご紹介します。

### 基本

- HTTP & HTTPS 対応
- BitTorrent、Magnet & ed2k 対応
- Seed、DHT、PEX、uTP、Webtorrent、Upnp
- トラッカーリストの毎日自動更新
- 多言語対応
- ダークモード

### 上級機能

基本的なダウンロード機能に加えて、Gopeedをさらに活用できる多くの高度な機能があります。

- **オープン HTTP API**

  RESTFul API を通じて、Gopeedと簡単に連携できます。例えば、API を使って Gopeed のダウンロード、一時停止、削除などを制御できます。

- **非中央集権型拡張機能**

  `JavaScript` で拡張機能を記述し、Gopeed のダウンロード機能を強化できます。例えば、ウェブサイトから動画をダウンロードしたり、音楽をダウンロードしたりすることができます。

## 他のダウンローダーとの比較

市場で人気のあるダウンローダーとの比較を以下に示します。Gopeedの優位性が一目でわかります。

| HTTP                   | ✔️     | ✔️                          | ✔️                |
| BitTorrent             | ✔️     | ✔️                          | ❌                |
| Magnet                 | ✔️     | ✔️                          | ❌                |
| クロスプラットフォーム対応 | ✔️     | ❌ （デスクトップのみ）         | ❌ （Windowsのみ） |

| オープンソース           | ✔️     | ✔️                          | ❌                |
| 非 Electron            | ✔️     | ❌                          | ✔️                |
| オープン API            | ✔️     | ✔️                          | ❌                |
| 拡張機能対応            | ✔️     | ❌                          | ❌                |

もちろん、Gopeedにはまだ多くの改善点があるかもしれませんが、継続的に改善していく予定です。

また、Gopeedはオープンソースですので、いつでもどこでもアイデアを提案したり、直接コードを貢献して Gopeed をより良くすることができます。
