Gopeed는 외부 사용을 위한 HTTP API 인터페이스를 제공하여, API를 통해 다운로드를 관리할 수 있습니다.

## API 활성화

먼저 통신 프로토콜을 TCP로 설정해야 합니다. **설정 -> 고급 -> 통신 프로토콜**로 이동하여 통신 프로토콜을 TCP로 설정하세요. 그런 다음 아래 이미지와 같이 IP와 포트를 설정하세요:

![](/images/dev/set-port.png)

<Callout type="info">
  원하는 `port`를 설정할 수 있지만, 이미 사용 중이거나 시스템에서 예약된 포트는 사용하지 마세요. 접속 실패가 발생할 수 있습니다.
</Callout>

그 후 `http://127.0.0.1:9999`를 통해 API에 접근할 수 있습니다. 보안상의 이유로 토큰을 설정하는 것을 권장합니다. **설정 -> 고급 -> API 토큰**으로 이동하여 아래 이미지와 같이 토큰을 설정하세요:

![](/images/dev/set-token.png)

> 참고: 위 설정은 재시작 후에 적용됩니다.

## JS SDK 사용

Gopeed는 공식 JS 라이브러리를 제공합니다. 먼저 `npm install @gopeed/rest`를 실행하여 설치해야 합니다:

```
npm install @gopeed/rest
```

그런 다음 자유롭게 사용할 수 있습니다. 예를 들면:

```js
import { Client } from "@gopeed/rest";

(async function () {
  // 클라이언트 생성
  const client = new Client();
  // 작업 생성
  const res = await client.createTask({
    req: {
      url: "https://example.com/file.zip",
    },
  });
})();
```

더 많은 사용법, 소스 코드 및 패키지 업데이트는 [공식 Gopeed JS 저장소](https://github.com/GopeedLab/gopeed-js)에서 확인할 수 있습니다.
