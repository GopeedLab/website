Gopeed(전체 이름 Go Speed)는 `Golang` + `Flutter`로 개발된 고속 다운로더입니다. `HTTP`, `BitTorrent`, `Magnet`, `ed2k` 다운로드를 지원하며, 다음을 포함한 모든 플랫폼을 지원합니다:

- [x] **Windows**
- [x] **Macos**
- [x] **Linux**
- [x] **Android**
- [x] **iOS**
- [x] **Web**
- [x] **Docker**
- [x] **Command Tool**

## 기능

최신 다운로더로서 Gopeed는 다양한 기능을 제공합니다. 간단히 소개해 드리겠습니다.

### 기본 기능

- HTTP 및 HTTPS 지원
- BitTorrent, Magnet 및 ed2k 지원
- Seed, DHT, PEX, uTP, Webtorrent, Upnp
- 매일 자동으로 트래커 목록 업데이트
- 다국어 지원
- 다크 모드

### 고급 기능

기본 다운로드 기능 외에도 Gopeed를 더욱 활용할 수 있는 다양한 고급 기능이 있습니다.

- **공개 HTTP API**

  RESTFul API를 통해 Gopeed와 쉽게 상호 작용할 수 있습니다. 예를 들어, API를 통해 Gopeed의 다운로드, 일시 정지, 삭제 등을 제어할 수 있습니다.

- **탈중앙화 확장 프로그램**

  `JavaScript`로 확장 프로그램을 작성하여 Gopeed의 다운로드 기능을 향상시킬 수 있습니다. 예를 들어 웹사이트에서 비디오를 다운로드하거나 음악을 다운로드하는 등의 작업이 가능합니다.
