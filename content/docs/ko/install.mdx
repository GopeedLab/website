다음 방법으로 모든 플랫폼에서 Gopeed를 빠르게 다운로드할 수 있습니다:

1. [공식 웹사이트](https://gopeed.com)에서 다운로드하면 운영체제에 맞는 버전이 자동으로 선택됩니다.
2. [GitHub](https://github.com/GopeedLab/gopeed/releases)에서 직접 원하는 버전을 선택하여 다운로드할 수 있습니다.

## 플랫폼별 설치 가이드

### Windows

Windows는 두 가지 배포판이 있습니다. 하나는 포터블 버전이고, 다른 하나는 설치 버전입니다. 포터블 버전은 설치할 필요 없이 압축을 풀고 실행하면 됩니다. 설치 버전은 설치가 필요하며, 설치 과정은 다른 소프트웨어와 동일합니다. 본인의 취향에 따라 선택하시면 됩니다.

> 포터블 버전의 파일명은: Gopeed-v1.x.x-windows-amd64-portable.zip

직접 다운로드하는 것 외에도 패키지 관리자를 통해 Gopeed를 설치하거나 업그레이드할 수 있습니다:

- [Scoop](https://github.com/ScoopInstaller/Scoop): 커뮤니티에서 관리하는 Windows용 명령줄 패키지 관리자입니다. 가볍고 유연하며, 관리자 권한 없이 설치할 수 있고 소프트웨어 관리와 업데이트가 간편합니다.

  ```powershell
  scoop bucket add extras
  scoop install extras/gopeed
  ```

- [Winget](https://learn.microsoft.com/ko/windows/package-manager/winget): Microsoft의 공식 패키지 관리자로, Windows 운영체제와 깊게 통합되어 있습니다.

  ```powershell
  winget install -e --id monkeyWie.Gopeed
  ```

#### 방화벽 오탐지

Windows 시스템에서 일부 백신 소프트웨어가 Gopeed의 두 핵심 구성 요소를 오탐지할 수 있습니다: 브라우저 확장 프로그램 통신용 `host.exe`와 애플리케이션 업데이트용 `updater.exe`입니다. 브라우저 확장 프로그램이 다운로드 작업을 올바르게 가로채고 애플리케이션이 자동 업데이트될 수 있도록 `Gopeed 설치 디렉토리`를 방화벽이나 백신 소프트웨어의 화이트리스트에 추가해 주세요.

<Callout type="info">
**안전한 다운로드**: 반드시 이 문서에 설명된 방법으로 Gopeed를 다운로드하세요. [공식 웹사이트](https://gopeed.com), [공식 GitHub 저장소](https://github.com/GopeedLab/gopeed/releases), 또는 Scoop이나 Winget을 통해서만 다운로드하세요. 소프트웨어의 보안을 위해 제3자 출처는 사용하지 마세요.

**오탐지에 대하여**: 이는 Go 언어로 컴파일된 애플리케이션의 특성과 관련이 있을 수 있으며, 현재 좋은 해결책이 없습니다. 향후 구성 요소를 Rust로 마이그레이션하여 이 문제를 개선할 계획입니다.

</Callout>

### Linux

Linux 시스템에서는 `.deb` 및 `.AppImage` 패키지를 제공하므로, 가장 적합한 것을 선택하시면 됩니다. 또한 다음 명령어로 빠르게 설치할 수도 있습니다:

- Flatpak

  ```sh
  flatpak install flathub com.gopeed.Gopeed
  ```

- Snap

  ```sh
  sudo snap install gopeed
  ```

### macOS

macOS 버전은 `.dmg` 파일을 제공하며, 더블클릭하여 설치할 수 있습니다. 설치 패키지는 `intel` 및 `apple silicon` 두 가지 아키텍처를 지원합니다.

> 팁: macOS에서 실행이 실패하는 경우, 터미널에서 `xattr -d com.apple.quarantine /Applications/Gopeed.app` 명령을 실행해 주세요

### Android

Android 버전은 `.apk` 파일을 제공하며, 직접 다운로드하여 설치할 수 있고 모든 CPU 아키텍처를 지원합니다.

### iOS

현재 iOS 플랫폼에서는 `.ipa` 파일만 제공되며, 자체 서명 후 설치해야 합니다. [TrollStore](https://github.com/opa334/TrollStore)를 사용하여 설치하는 것을 권장합니다.

> App Store에 등록되지 않은 이유는 무엇인가요?
>
> Apple의 심사 메커니즘으로 인해 BitTorrent 프로토콜 관련 앱은 등록이 허용되지 않습니다.
>
> TestFlight에 등록되지 않은 이유는 무엇인가요?
>
> 돈이 없기 때문입니다! 이 프로젝트는 순수하게 열정으로 운영됩니다. 충분한 후원을 받을 수 있다면 즉시 TestFlight에 등록하겠습니다. 그러니 많은 후원 부탁드립니다. 오픈 소스는 쉽지 않습니다. 감사합니다!

### Web

원격 다운로드 서비스가 필요한 경우 Web 버전을 사용해 보세요. Gopeed는 다양한 플랫폼용 Web 버전을 제공합니다. 시스템 및 CPU 아키텍처에 맞는 버전을 다운로드하시면 됩니다.

![web](/images/guide/install/web.png)

다음은 Windows 플랫폼에서 Web 버전을 사용하는 예시이며, 다른 플랫폼에서도 사용법은 비슷합니다.

1. Web 버전을 다운로드하고 압축을 풀면 폴더가 생성됩니다. 원하는 위치에 배치하세요.
2. 해당 폴더의 루트 디렉토리에서 터미널을 열고 `./gopeed.exe`를 실행하세요. 성공적으로 실행되면 다음과 같은 출력이 표시됩니다:

   ```bash
      _______   ______   .______    _______  _______  _______
    /  _____| /  __  \  |   _  \  |   ____||   ____||       \
    |  |  __  |  |  |  | |  |_)  | |  |__   |  |__   |  .--.  |
    |  | |_ | |  |  |  | |   ___/  |   __|  |   __|  |  |  |  |
    |  |__| | |  `--'  | |  |      |  |____ |  |____ |  '--'  |
    \______|  \______/  | _|      |_______||_______||_______/

    Server start success on http://[::]:9999
   ```

3. 브라우저를 열고 `http://localhost:9999`에 접속하세요.

#### Web 설정

Web 버전은 명령줄 매개변수 또는 설정 파일을 통한 구성을 지원합니다. 명령줄 매개변수는 `./gopeed.exe -h`로 확인할 수 있습니다:

```sh
$ ./gopeed.exe -h
Usage of C:\Users\levi\Downloads\gopeed-web-v1.3.13-windows-amd64\gopeed.exe:
  -A string
        Bind Address (default "0.0.0.0")
  -P int
        Bind Port (default 9999)
  -T string
        API token, that can only be used when basic authentication is enabled.
  -c string
        Config file path (default "./config.json")
  -d string
        Storage directory
  -p string
        HTTP Basic Auth Password
  -u string
        HTTP Basic Auth Username (default "gopeed")
```

또한 설정 파일을 통한 구성도 지원합니다. 루트 디렉토리에 `config.json` 파일을 생성하고 다음 내용을 입력하세요:

```json
{
  "address": "",
  "port": 0,
  "username": "",
  "password": "",
  "apiToken": "",
  "storageDir": "",
  "whiteDownloadDirs": ["/root/downloads", "/root/dir/*", "/root/dir?abc"]
}
```

환경 변수를 통한 구성도 지원합니다. 규칙은 `GOPEED_ConfigKey`이며, 예시는 다음과 같습니다:

```sh
export GOPEED_ADDRESS="0.0.0.0"
export GOPEED_PORT="9999"
export GOPEED_USERNAME="gopeed"
export GOPEED_PASSWORD="xxx"
export GOPEED_APITOKEN=""
export GOPEED_STORAGEDIR=""
export GOPEED_WHITEDOWNLOADDIRS="/root/downloads,/root/dir/*,/root/dir?abc"
```

<Callout type="warn">
  공인 IP에 배포하는 경우, 반드시 신원 인증을 활성화하세요. 그렇지 않으면 보안 위험이 있습니다.
</Callout>

#### 기본 다운로드 설정

서버가 처음 시작될 때 기본 다운로드 설정을 지정해야 하는 경우, `config.json`에 `downloadConfig` 필드를 추가할 수 있습니다. 설정 세부 사항은 [DownloaderStoreConfig](https://pkg.go.dev/github.com/GopeedLab/gopeed/pkg/base#DownloaderStoreConfig)를 참조하세요. 다음은 예시입니다:

```json
{
  "address": "127.0.0.1",
  "port": 9999,
  "downloadConfig": {
    "downloadDir": "d:/test"
  }
}
```

### Docker

한 줄 명령어:

```sh
docker run --name gopeed -d -p 9999:9999 liwei2633/gopeed
```

다운로드 디렉토리 마운트

```sh
docker run --name gopeed -d -p 9999:9999 \
    -v /path/to/download:/app/Downloads \
    liwei2633/gopeed
```

데이터 디렉토리 마운트

```sh
docker run --name gopeed -d -p 9999:9999 \
    -v /path/to/download:/app/Downloads \
    -v /path/to/storage:/app/storage liwei2633/gopeed
```

컨테이너 그룹 ID 및 사용자 ID 지정

```sh
docker run --name gopeed -d -p 9999:9999 \
    -e PGID=100 \
    -e PUID=1000 \
    liwei2633/gopeed
```

신원 인증을 활성화해야 하는 경우 환경 변수 매개변수를 전달할 수 있습니다:

```sh
docker run --name gopeed -d -p 9999:9999 \
    -e GOPEED_USERNAME="admin" \
    -e GOPEED_PASSWORD="123" \
    -v /path/to/download:/app/Downloads \
    -v /path/to/storage:/app/storage \
    liwei2633/gopeed
```

### 명령줄

명령줄 버전은 `Golang` 환경이 필요합니다. `Golang` 환경이 설치되어 있지 않은 경우 [여기](https://golang.org/doc/install)를 참조하여 설치하세요.

설치 명령어:

```sh
go install github.com/GopeedLab/gopeed/cmd/gopeed@latest
```

설치 후 `gopeed` 명령어로 다운로드할 수 있습니다. `gopeed -h`로 구체적인 사용법을 확인할 수 있습니다:

```sh
$ gopeed -h
Usage of gopeed:
  -C int
        Concurrent connections. (default 16)
  -D string
        Store directory. (default "C:\\Users\\levi")
```

#### 명령줄 사용 예시

HTTP 리소스 다운로드:

```sh
gopeed https://example.com/file.zip
```

토렌트 파일 다운로드:

```sh
gopeed D:/Downloads/file.torrent
```

마그넷 링크 다운로드:

```sh
gopeed magnet:?xt=urn:btih:xxxx
```

ed2k 링크 다운로드:

```sh
gopeed 'ed2k://|file|example.iso|123456789|0123456789ABCDEF0123456789ABCDEF|/'
```
