O Gopeed fornece uma interface de API HTTP para uso externo, permitindo o gerenciamento de downloads através da API.

## Habilitar API

Primeiro, você precisa definir o protocolo de comunicação como TCP. Vá em **Configurações → Avançado → Protocolo de Comunicação** e defina o protocolo de comunicação como TCP. Em seguida, defina o IP e a porta conforme mostrado na figura abaixo:

![](/images/dev/set-port.png)

<Callout type="info">
  Você pode definir qualquer `porta`, mas cuidado para não usar portas que já estejam em
  uso ou reservadas pelo sistema, pois isso pode causar falhas de acesso.
</Callout>

Depois disso, você pode acessar a API através de `http://127.0.0.1:9999`. Por motivos de segurança, é recomendável definir um token. Vá em **Configurações → Avançado → API Token** e defina um token conforme mostrado na figura abaixo:

![](/images/dev/set-token.png)

> Nota: As configurações acima entram em vigor após uma reinicialização.

## Usando o SDK JS

O Gopeed fornece uma biblioteca JS oficial, que precisa ser instalada primeiro executando `npm install @gopeed/rest`:

```
npm install @gopeed/rest
```

Depois você pode usá-la tranquilamente, por exemplo:

```js
import { Client } from "@gopeed/rest";

(async function () {
  // Criar um cliente
  const client = new Client();
  // Criar uma tarefa
  const res = await client.createTask({
    req: {
      url: "https://example.com/file.zip",
    },
  });
})();
```

Mais usos, código-fonte e atualizações do pacote estão disponíveis no [repositório oficial do Gopeed JS](https://github.com/GopeedLab/gopeed-js).
