Gopeed (nome completo Go Speed) é um downloader de alta velocidade desenvolvido em `Golang` + `Flutter`, que suporta downloads via `HTTP`, `BitTorrent`, `Magnet` e `ed2k`, e é compatível com todas as plataformas, incluindo:

- [x] **Windows**
- [x] **Macos**
- [x] **Linux**
- [x] **Android**
- [x] **iOS**
- [x] **Web**
- [x] **Docker**
- [x] **Ferramenta de Linha de Comando**

## Funcionalidades

Como um downloader moderno, o Gopeed possui muitos recursos. Aqui está uma breve introdução.

### Básico

- Suporte a HTTP e HTTPS
- Suporte a BitTorrent, Magnet e ed2k
- Seed, DHT, PEX, uTP, Webtorrent, Upnp
- Atualização automática da lista de trackers todos os dias
- Internacionalização
- Modo escuro

### Avançado

Além das funções básicas de download, existem muitos recursos avançados que tornam o Gopeed mais versátil.

- **API HTTP aberta**

  Através da API RESTful, você pode interagir facilmente com o Gopeed. Por exemplo, pode controlar o download, pausar, excluir, etc. através da API.

- **Extensão descentralizada**

  Você pode escrever extensões através de `JavaScript` para aprimorar os recursos de download do Gopeed, como baixar vídeos de um site ou baixar músicas de um site, etc.
