Gopeed (полное название Go Speed) — это высокоскоростной менеджер загрузок, разработанный на `Golang` + `Flutter`, который поддерживает загрузку по протоколам `HTTP`, `BitTorrent`, `Magnet` и `ed2k`, а также работает на всех платформах, включая:

- [x] **Windows**
- [x] **Macos**
- [x] **Linux**
- [x] **Android**
- [x] **iOS**
- [x] **Web**
- [x] **Docker**
- [x] **Командная строка**

## Возможности

Как современный менеджер загрузок, Gopeed обладает множеством возможностей. Ниже приведено краткое описание.

### Основные

- Поддержка HTTP и HTTPS
- Поддержка BitTorrent, Magnet и ed2k
- Seed, DHT, PEX, uTP, Webtorrent, Upnp
- Автоматическое ежедневное обновление списка трекеров
- Интернационализация
- Тёмная тема

### Расширенные

Помимо базовых функций загрузки, существует множество расширенных возможностей, которые делают Gopeed более гибким.

- **Открытый HTTP API**

  Через RESTful API вы можете легко взаимодействовать с Gopeed, например, управлять загрузками, паузой, удалением и т. д. через API.

- **Децентрализованные расширения**

  Вы можете писать расширения на `JavaScript` для расширения возможностей загрузки Gopeed, например, загружать видео с веб-сайтов или скачивать музыку и т. д.
